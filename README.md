# 🌟 Dummy-Shopping-Site

&#x20; &#x20;

## 🚀 Описание проекта

Простой интернет магазин

## 🔧 Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/Yoko-ship/shopping-site
cd repository

# Установка зависимостей
npm install

# Запуск проекта
npm start
```


## 🛠 Используемые технологии

- 🟢 React
- 🎨 NextJS
- 🖼️ TypeScript
